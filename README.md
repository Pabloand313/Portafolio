# Portafolio Profesional - Desarrollo Web Front-End

## Descripción del Proyecto
Este proyecto consiste en un sitio web de una sola página diseñado como un portafolio profesional para presentar mi perfil como estudiante de Ingeniería en Informática, el sitio integra información personal, habilidades técnicas, proyectos realizados y un formulario de contacto, cumpliendo con los estándares de semántica HTML5 y diseño responsivo 

## Tecnologías Utilizadas
* **HTML5:** Uso de etiquetas semánticas para una estructura accesible y organizada (`<header>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`)
* *CSS3:** Implementación de estilos modernos utilizando **Flexbox** para la navegación y componentes, y **CSS Grid** para la galería de proyectos
* **Diseño Responsivo:** Adaptación a diferentes dispositivos (Desktop, Tablet y Mobile) mediante el uso de **Media Queries**
* **Git & GitHub:** Control de versiones con un flujo de trabajo basado en commits descriptivos

## Estructura del Sitio
El portafolio se divide en las siguientes secciones clave:
1.  **Header:** Contiene el logo personal y una barra de navegación con enlaces internos (anclas)
2.  **Sobre Mí:** Breve biografía profesional, datos personales y una cita inspiradora
3.  **Habilidades:** Tabla detallada con niveles de dominio de tecnologías y listas de competencias técnicas
4.  **Proyectos:** Galería visual que utiliza CSS Grid para mostrar trabajos previos con enlaces a sus respectivos repositorios
5.  **Contacto:** Formulario funcional con validación nativa de HTML5 para capturar mensajes de usuarios
6.  **Footer:** Enlaces a redes sociales y créditos de autoría
